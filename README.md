# Modern Responsive Dashboard

A sleek and functional dashboard interface built with **Vanilla JavaScript**, **HTML5**, and **CSS3**. This project focuses on modern layout techniques and a seamless user experience with theme customization.

## 🌟 Key Features
- **Fully Responsive Design:** Optimized for all screen sizes (Mobile, Tablet, and Desktop).
- **Dark & Light Mode:** User-toggleable themes with persistent settings.
- **Modern Layouts:** Built using **CSS Grid** and **Flexbox** for a robust and flexible structure.
- **Interactive UI:** Dynamic elements powered by clean JavaScript.

## 🛠️ Built With
- **HTML5:** Semantic structure.
- **CSS3:** Advanced styling, including **Grid System**, **Flexbox**, and CSS Variables for theming.
- **JavaScript (ES6):** Logic for theme switching and interactive components.
